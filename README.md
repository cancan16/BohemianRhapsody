## 使用Java开发公链与交易所项目

### 1.项目介绍
通过Java Springboot开发一个简易版公链项目，并实现钱包功能，再通过SpringCloud开发一套中心化交易所项目，考虑再用Flutter实现移动端的APP，API则通过[yapi][yapi]进行管理。

### 2.使用技术
1. Java Spring Boot  
2. Java Spring Cloud  
3. Flutter
